# typical90

## Overview

RustでAtCoderの[競プロ典型90問](https://atcoder.jp/contests/typical90/tasks)を解いています。その回答をこのリポジトリに残しています。

まずは★2~★4を埋めるのが目標です。

## Requirement

- wsl2
- cargo
- proconio 0.3.6
- ndarray 0.13.0
